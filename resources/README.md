# Recursos do Aplicativo

Para gerar os ícones e splash screens para Android e iOS, você deve adicionar os seguintes arquivos nesta pasta:

1.  **icon.png**: Ícone do aplicativo (1024x1024px).
2.  **splash.png**: Tela de abertura (2732x2732px).

Após adicionar os arquivos, execute o comando para gerar os recursos nativos (se estiver usando a CLI do Capacitor/Cordova Res):
`npx capacitor-assets generate`
