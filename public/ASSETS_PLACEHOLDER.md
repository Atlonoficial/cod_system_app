# Assets do Student App

Para personalizar o Student App, substitua os seguintes arquivos nesta pasta:

1.  **logo.png**: Logo principal do aplicativo.
2.  **favicon.ico**: Ícone de favorito para o navegador.
3.  **og-image.png**: Imagem para compartilhamento em redes sociais (Open Graph).
4.  **manifest.json**: Atualize o nome e ícones se necessário.

Certifique-se de que os nomes dos arquivos correspondam aos referenciados no código ou atualize o código para apontar para os novos arquivos.
